1️⃣ Difference between var, let, and const

ans::var: Function-scoped; can be redeclared and updated.

let: Block-scoped; can be updated but not redeclared in the same scope.

const: Block-scoped; cannot be updated or redeclared.

2️⃣ Spread Operator (...)
ans::Used to expand or copy elements of an array or properties of an object.

3️⃣ Difference between map(), filter(), and forEach()

ans::map(): Creates a new array after transforming each element.

filter(): Returns elements that satisfy a condition.

forEach(): Iterates through each element without returning a new array.

4️⃣ Arrow Function
ans::A shorter syntax for writing functions in JavaScript.

5️⃣ Template Literals
ans::A way to write strings that allows embedded expressions and multi-line text using backticks.
